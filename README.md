# SCT_WD_3
Quiz Game Application

A modern and interactive Quiz Game Application built using HTML, CSS, and JavaScript. The project features a futuristic dark neon-themed UI, multiple question formats (Single Select, Multi Select, and Fill in the Blank), a real-time countdown timer, progress tracking, score calculation, performance analytics, and detailed answer review. Designed with a responsive and engaging user experience, this application demonstrates core frontend development skills, DOM manipulation, event handling, and dynamic UI rendering.
